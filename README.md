# ツキフダ 〜月夜の十二番勝負〜

**▶ 今すぐ遊ぶ: https://nus-coder.github.io/tsukifuda/**

同じ12枚の妖怪カードを持ち寄り、月光点を奪い合う2人対戦カードゲームのWebアプリ。

- **CPU対戦**：見習い妖怪（初心者向け）／大妖怪（ハード、後悔マッチングによる混合戦略AI）
- **ストーリーモード**：「月夜の五番勝負」— 個性とギミックを持つ5人のボスを順に倒す一人用モード
- **オンライン対戦**：合言葉コードで友達と1対1（PeerJS / WebRTC、サーバー実装不要）＋絵文字リアクション
- 効果音（Web Audio合成）・カード固有演出・ページ式チュートリアル・中断メニューつき
- 1ゲーム10〜20分。ルールは3分で覚えられる。

## 遊び方（すぐ始める）

ビルド不要。静的サーバーで配信するだけ。

```bash
cd tsukifuda
python3 -m http.server 8347
# → http://localhost:8347 を開く
```

`index.html` を直接開いても動きますが、オンライン対戦（PeerJS CDN）を使う場合は
http(s) で配信するのが確実です。GitHub Pages / Vercel / Netlify などにそのまま置けます。

## ルール概要

- 両者が**まったく同じ12枚**（パワー0〜11、固有能力つき）を手札に持ち、12ラウンド戦う。
- 各ラウンドの得点（月光点）は「月齢カード」で決まり、**12ラウンド分すべて最初から公開**。
- 毎ラウンド同時に1枚出し。勝者が月光点を取る。引き分けは**持ち越し**で賭け金が膨らむ。
- 使ったカードは戻らない＝**相手の残り手札は常に丸見え**。純粋な読み合いのゲーム。
- 新月（弱い方が勝つ）・月蝕（能力無効）・満月（人狼強化）が戦略を反転させる。

詳細ルール・カード一覧・処理順は [docs/SPEC.md](docs/SPEC.md) を参照。
アプリ内の「遊び方」にも同じ内容を収録。

## 開発

```bash
node test/engine.test.js   # エンジン＋AIのユニットテスト
```

| ファイル | 役割 |
|---|---|
| `js/engine.js` | ゲームロジック（純粋関数、UI/AI/通信に非依存） |
| `js/cards.js` | カード定義＋SVGイラスト＋月齢定義 |
| `js/ai.js` | CPU（初心者=価値マッチング＋ノイズ / ハード=利得行列＋後悔マッチング） |
| `js/ui.js` | DOM描画・演出 |
| `js/online.js` | PeerJS接続とコミット＆リビール（チート防止） |
| `js/main.js` | 画面遷移・進行管理 |

今後の拡張アイデアは [docs/IDEAS.md](docs/IDEAS.md) にまとめてある。
